Documentation

A reference page for documentation, data sources, workflow details, and licensing information.

Workflow

This project combines mobility and environmental data from different sources. The workflow involved several key steps:

  • Data collection.
  • Data cleaning and tranformation.
  • Visualisation.

The code used to process the data, generate the visualizations and all other relevant documentation is available in our githubrepository.

Conclusions

Through this project, we explored trends in vehicle composition and air quality in Milan and Bologna. The visualizations help reveal patterns and variations over time, offering valuable insights for further research or policy discussions.
