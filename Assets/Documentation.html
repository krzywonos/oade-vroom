Documentation

A reference page for documentation, data sources, workflow details, and licensing information.

Workflow

This project combines mobility and environmental data from different sources, including datasets from the cities of Bologna and Milan. The workflow involved several key steps:

  • Data collection. We retrieved datasets from public open data portals and institutional sources, covering vehicle registrations, air quality, and public transport statistics.
  • Data cleaning and transformation. We processed the data to ensure consistency across sources, removed duplicates, handled missing values, and aggregated the data where necessary to match the visualisation needs.
  • Visualisation. We created a series of interactive visualisations tailored to the type of data — including bar charts, line charts, and pie charts — to allow users to explore trends in mobility and air quality across both cities.

The code used to process the data, generate the visualizations and all other relevant documentation is available in our github repository.

Conclusions

This project offers an integrated view of urban mobility and environmental trends in Bologna and Milan. By combining and visualising heterogeneous datasets, we provide users with an accessible way to explore patterns in vehicle composition, air quality, and public transport usage over time. The flexibility of the visualisations allows for dynamic exploration of the data, supporting both citizen awareness and potential policy discussions. Future developments may include the integration of additional datasets and the refinement of interactive features to further enhance the user experience.
